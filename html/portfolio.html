Musicify

Musicify provides a method for a user to find new music based on their established interests. Specifically, the user provides the name of an artist whose music they enjoy, and Musicify implements multiple API's to dynamically generate a Spotify playlist of songs by similar artists. While Spotify and similar services have features that generate playlists bases on listener data, Musicify allows the user play a more direct role in finding new music.

Animal Apparitions

Animal Apparitions is an application that took root as an exercise in using AJAX calls to render specific JSON information based on user input. In this case, the user clicks on existing buttons or dynamically generates a new one and extracts ten GIF's that correspond to the user's input.

Crystals Collector

Crystals Collector stemmed from an activity wherein JQuery is utilized to facilitate Data Object Model manipulation in accordance with user interactions. In this example, the user is given a random number and four crystals to choose from. Each crystal has a random numeric value. The point of the game is to click the crystals to accumulate points until your score equals the given random number (deemed a "Win") or exceeds it (a "Loss").

Psychic Game

The Psychic Game primarily makes use of JavaScript to expediate complex functionality based on user input. Here, the computer randomly selects a number and the user guesses what the number is. A scoreboard displays a record of the user's guesses and restarts the game upon a win, where the user correctly guesses the number, or after a loss, consisting of 9 failed attempts.
